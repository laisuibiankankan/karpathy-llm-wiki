# 漏油排查原理

> 基于32条漏油source页的实际排查路径归纳。

## 第一步：先分清真假漏油

**原理**：转向机出厂时输入轴处涂有密封脂/防锈油，受热后会变成油状液体渗出。同时低温环境下油脂也可能凝结后化开造成渗油假象。约16%的漏油反馈最终确认为假漏油。

**判断方法**（来自案例实际操作）：

1. **去防护盖** — 取下输入轴保护壳/防护罩，暴露漏油区域
2. **擦干净** — 将疑似漏油位置的油液彻底擦净
3. **冷启动验证** — 无需转动方向盘，确认是否直接漏油（案例#20）
4. **热车后验证** — 慢速转动方向盘至转向机表面温热（约10分钟以上），再擦干净观察（案例#20）
5. **运行观察** — 原地打方向或短途行驶后，观察5-15分钟是否再次渗油

**判断依据**：
- 不再渗 = 假漏油（密封脂/防锈油/低温凝结油脂）
- 继续渗 = 真漏油（密封件损坏）

> **案例佐证**：#54（三一，疑似防锈油，擦净后未再漏）、#187（东风，润滑脂变稀，清理后未再漏）、#20（乌鲁木齐低温，擦净后无漏油）、#62（宇通X国，-40℃侧盖，擦净后未再漏）

## 第二步：确认漏油后，定位漏油点

### 核心排查路径

```
漏油报告
  │
  ├─ 去除防护壳/罩
  │
  ├─ 擦净油液 → 原地打方向观察
  │     │
  │     ├─ 不再渗 → 确认假漏油 ✓
  │     │     └─ 告知客户正常现象，车辆放行
  │     │
  │     └─ 持续渗 → 真漏油，进入分步定位
  │           │
  │           ├─ 查看进油口/回油管接头
  │           │     ├─ 接头松动/垫片损坏 → 紧固或更换 (#22、#99)
  │           │     └─ 接头无异常 → 转到下一级
  │           │
  │           ├─ 输入轴区域
  │           │     ├─ 轴密封与壳体接合处有油 → 轴密封损伤
  │           │     ├─ 铁丝堵处有油迹 → 铁丝堵磕碰 (#135)
  │           │     └─ 单独查看输入轴密封唇口 → 唇口损伤/异物 (#5、#81、#98、#131、#250)
  │           │
  │           ├─ 输出轴区域
  │           │     ├─ 摇臂轴密封圈处 → 密封老化磨损 (#36、#110、#139)
  │           │     ├─ 侧盖处 → 侧盖密封不良 (#62)
  │           │     └─ 泄压阀处 → 泄压阀O型圈失效 (#48)
  │           │
  │           ├─ 限位螺栓处
  │           │     └─ 螺栓脱出或松动 → 装配问题（滑套浅/漏装）(#175、#201)
  │           │
  │           ├─ 法兰轴与壳体接合面
  │           │     └─ 结合面渗油 → 密封不良 (#75)
  │           │
  │           └─ 壳体本体
  │                 └─ 疑似沙眼 → 擦净打方向验证 (#198)
  │
  └─ 辅助手段
        ├─ 喷显影剂（案例#139用显影剂确认漏油点）
        ├─ 油罐液位检查（油液减少辅助确认漏油）
        ├─ 视频诊断（大量案例使用：查看漏油位置、流量、色泽）
        └─ 在保→换件，旧件返回实验室分析
```

### 视频诊断要点（来自实际案例经验）

多个案例（#103、#109、#119、#139、#203、#211、#235）均采用视频连线确认漏油：
- 要求拍摄**多角度**：漏油点近景、油管接头、防护盖完整度
- 检查**防护盖是否完整**（#91防护盖缺失一小块→沙尘进水风险）
- 确认**油液是否明显减少**（油罐液位是辅助判断依据）
- 拆下防护盖后**转动方向**观察动态漏油

## 第三步：分阶段判断原因

### 0km（新车/新装）漏油 → 装配线/供应链问题

**特征**：刚出厂/刚装车就发现漏油，无明显使用痕迹

**常见根因**（来自source页）：
1. **轴密封来料损伤**（#5、#98）— KACO供应的轴封有初始损伤
2. **轴密封处异物**（#131、#250）— 塑料类异物卡在密封面
3. **限位螺栓未装到位/漏装**（#77、#97、#175、#201）— 装配防错失效
4. **铁丝堵磕碰**（#135）— 客户装配过程磕碰导致
5. **防锈油残留被误判**（#54）— 正常现象

**排查建议**：
- 务必先擦净验证（排除假漏油）
- 检查螺纹端面是否已喷漆、螺纹是否生锈（判断装配时是否有零件）
- 返回实验室拆解确认密封件状态

### field（使用后）漏油 → 使用/环境/磨损

**特征**：使用了数月至数年后开始漏油

**常见根因**（来自source页）：
1. **密封件长期磨损**（#81行驶8678km轴密封唇口损伤、#211约17万km）
2. **低温环境**（#20乌鲁木齐、#69黑龙江、#211乌鲁木齐—均低于-25℃时密封圈回弹变差）
3. **高温老化**（#148比亚迪西班牙—确认为高温导致漏油）
4. **外部干涉/磕碰**（#91防护盖缺失）
5. **来料缺陷迟发失效**（#48泄压阀来料问题）

**排查建议**：
- 询问车辆使用环境（最低温度、风力/风沙、路况）
- 查看保养记录（油液品牌、更换周期）
- 确认是否批量发生（#110、#148为批量漏油）
- 在保→换件+旧件分析；出保→友好协商

## 特别提醒

### 常见排查陷阱

1. **不拆防护盖直接判断** — 防护盖内常有正常油脂积聚，未去除直接看易误判
2. **冷车启动立即观察** — 低温下密封回弹慢，可能暂时不漏；需热车后测试
3. **一次擦净不够** — 建议擦净→运行→再擦净→再观察，重复确认
4. **忽略油管接头** — #99案例中大量油液从输入轴附近流出，实为进油口接头问题

### 特殊情况处理

- **出保车辆漏油**（#110、#142）：确认漏油（是否真漏）→客户自费或友好协助
- **批量漏油**（#110已出保批量、#148在保批量）：需调取车辆信息、使用工况、油液品牌后综合判断
- **客户不配合退旧件**（#103成都公交、#147澳大利亚）：尽量争取分析件，但不应因此拒绝服务

---

> **一句话总结**：先擦净打方向分真假，再定位查密封/接头/螺栓，0km重点看装配，field重点看磨损+环境。
