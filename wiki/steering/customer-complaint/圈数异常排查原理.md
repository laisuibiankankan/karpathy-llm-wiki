# 圈数异常排查：原理一句话说清

## 左右圈数不对称 → 限位螺栓不对称

**原理**：转向机内部有限位螺栓，到位置了就卡住不让继续转。如果左右两根限位螺栓没调对称，就会一边多转半圈、一边少转半圈。

**怎么判断**：左打1.5圈，右打2.5圈（或反过来）。

**结论**：顶起前桥 → 调限位螺栓 → 5分钟解决。

## 方向打不到底 → 可能不是转向机的问题

**原理**：除了转向机自己的限位，车桥上还有独立的限位螺栓限制轮胎转角。如果桥限位先碰到了，转向机还能转但方向盘转不动了。

**怎么判断**：打到极限时轮胎和车架还有很大空间，但方向盘转不动了。

**结论**：调桥限位螺栓，不是转向机的事。

## 左右圈数不够 → 限位螺栓未压到位（换转向机后常见）

**原理**：新换转向机后，限位螺栓未完全压缩到工作位置，导致可转动范围变窄。顶起前桥左右打死，限位螺栓会被压缩到位。

**怎么判断**：更换转向机后总圈数明显变少（如从5圈变3圈），顶起前桥后又有改善空间。

**结论**：顶起前桥，不启动车辆，左右打死方向即可解决问题。

## EHPS圈数异常 → 先诊断标定角度

**原理**：电液转向机（EHPS）使用软件限位（软限位），标定角度决定了方向盘软件可转动的范围。如果出厂或更换转向机后标定角度不对，会出现圈数不足、左右不对称的问题。

**怎么判断**：圈数不足时桥限位还有较大空间；用诊断仪读取标定值，典型异常如左800°右900°（正常应720°左右对称）。

**结论**：用诊断仪重写标定值，标定时需监控前桥限位螺栓状态。

## 左右圈数不一致但桥限位有空间 → 限位螺栓过度压缩

**原理**：限位螺栓旋入过深会导致提前限位，转向角度变小；过度压缩则导致角度超大甚至回弹打手。

**怎么判断**：角度明显大于同型号其他车辆，或打到极限时方向盘回弹打手。

**结论**：更换/重新调整限位螺栓。

## 桥限位螺钉顶弯 → 转向机限位过度压缩

**原理**：如果转向机限位螺钉过度压缩，方向盘实际圈数大于转向机额定圈数（如3.1圈），导致桥限位承受异常压力而顶弯。

**怎么判断**：实车方向盘圈数 > 转向机额定圈数 → 转向机限位被过度压缩；实车圈数 < 转向机限位触发圈数 → 桥限位先受力。

**结论**：更换桥限位螺钉，调整转向机限位螺栓。

## 偶发车轮不转 → 检查油液

**原理**：极少数圈数相关异常与油液有关。液压系统中有空气或油液品质不佳时，可能出现偶发性动力传输中断。

**怎么判断**：偶发出现、非持续性；换油后故障消失。

**结论**：检查油液状态，必要时更换油液。

## 排故要点总结（流程图）

```
圈数角度异常
├─ 顶起前桥（悬空状态）
│  ├─ 左右打到极限
│  │  ├─ 碰到桥限位后恢复 → ③ EHPS? → 读标定值重新标定
│  │  │                         └─ HPS? → 限位螺栓调对称即可
│  │  └─ 未碰到桥限位 → 调桥限位螺栓
│  └─ 圈数明显变少 → 限位螺栓未压到位 → 左右打死压缩
├─ EHPS标定值异常 → 用诊断仪重新标定
└─ 回弹打手/角度过大 → 限位过度压缩 → 更换/重调限位螺栓
```
