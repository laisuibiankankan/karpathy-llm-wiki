# 间隙/跑偏排查：原理一句话说清

## 测间隙必须启动发动机 → 熄火时测不准

**原理**：熄火时液压系统不工作，转向机内部油压把阀门锁死在中间位置，这时候摇输入轴会觉得"间隙很大"。其实这是正常的液压锁定。

**怎么判断**：启动发动机 → 打方向 → 再测输入轴间隙。正常值 ≤5°（大约手指宽）。

**结论**：不启动就测间隙 = 白测、误诊。

## 跑偏先查轮胎和四轮定位 → 别急着动转向机

**原理**：车跑偏90%是轮胎气压不平衡、四轮定位不准、或者路面倾斜造成的。转向机让你跑偏的可能性很小。

**怎么判断**：平直路段松方向盘 → 车往一侧跑。

**结论**：先打气、再定位、最后才怀疑转向机。

## EHPS跑偏 → 先检查0位标定

**原理**：电液转向机（EHPS）的软件0位标定如果偏差，转向机在直行位时会始终输出一个偏转力矩，导致车辆持续跑偏。典型偏差量约30°。

**怎么判断**：跑偏车辆方向盘在正中位置，但车轮不在直行方向；或方向盘偏一定角度时车轮才直行。

**结论**：用诊断仪找到车轮0位 → 重新标定EHPS 0位 → 调整方向盘。

## 方向自转 → 先看输入轴受没受过外力

**原理**：转向机自己不会"主动"自转。自转现象几乎100%是因为输入轴受过磕碰/外力冲击，导致阀芯变形、扭杆变形或销子断裂，使转向机失去了对中能力。

**怎么判断**：检查输入轴端面有无磕碰痕迹；断开万向节单独测试转向机是否仍自转。

**结论**：自转 = 受外力 → 更换转向机 → 排查运输/装配环节的风险。

## 方向盘抖动 → EHPS断开传感器 vs HPS整车排查

**原理（EHPS）**：电液转向机的扭矩转角传感器如果损坏（如模塑环断裂），会给ECU发送错误的驱动扭矩信号，导致在轻打方向时出现抖动。

**怎么判断**：断开传感器线束 → 抖动消失 → 连接后抖动再现 → 确认传感器故障。

**结论**：EHPS抖动先断传感器测试；HPS抖动查整车共振、前束、主销、球头、万向节。

## 回弹打手 → 限位螺钉过度压缩

**原理**：限位螺钉过度压缩时，打到极限时会产生撞击回弹。尤其当匹配的电机泵保护压力小于转向机限压阀压力时，转向机无法正常泄压，加剧回弹。

**怎么判断**：打到极限位置时方向盘回弹打手。

**结论**：更换/调整限位螺钉。

## 不回正 → 管柱干涉还是主销不对？

**原理**：转向不回正通常不在转向机本身。方向盘到管柱之间的干涉（如护套摩擦、万向节卡滞）或主销后倾角/内倾角不对都会导致不回正。

**怎么判断**：排除管柱干涉 → 检查主销参数 → 确认转向机出厂检测记录。

**结论**：排查管柱 → 排查主销 → 最后才回传转向机检测。

## 真间隙在哪儿？→ 输入轴-输出轴联动测试定位

**原理**：启动状态下转动输入轴，如果输出轴随之转动但轮胎纹丝不动，说明间隙在轮胎侧（拉杆球头、桥端连接等处），而不是转向机内部。

**怎么判断**：启动发动机，稳方向盘，另一人观察轮胎是否随输出轴转动。

**结论**：输入轴→输出轴联动位。轮胎不动 → 排查拉杆到前桥。输出轴不动 → 排查转向机内部。

## 排故要点总结（流程图）

```
间隙/跑偏/回正问题
├─ 间隙大
│  ├─ 熄火测的？ → 重新启动再测（多半是假间隙）
│  └─ 启动后测
│     ├─ ≤5° → 正常
│     └─ >5° → 联动测试定位：输出轴转轮胎不动→查拉杆桥端；反之查转向机
├─ 跑偏
│  ├─ EHPS → 先检查0位标定（约30°跑偏多半是标定问题）
│  ├─ 支起桥不自转 → 排除转向机问题
│  └─ 检查轮胎气压和四轮定位
├─ 方向自转
│  └─ 检查输入轴磕碰/检查传感器连接器 → 外力导致
├─ 不回正/回弹打手
│  ├─ 回弹打手 → 限位过度压缩（匹配检查电机泵压力）
│  └─ 不回正 → 管柱干涉 → 主销参数 → 转向机出厂记录
├─ 抖动
│  ├─ EHPS → 断开传感器线束测试
│  └─ HPS → 整车前束/主销/球头/管柱/共振排查
└─ 后桥摆动(RAS) → 获取完整报文分析
```
