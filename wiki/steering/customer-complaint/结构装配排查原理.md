# 结构/装配/损坏排查原理

> 基于28条结构装配source页的实际排查路径归纳。

## 第一步：确认损伤类型与来源

结构装配类问题多数在**进货检验**或**产线装配过程中**发现，关键是把问题分类到正确的源头，避免误判。

### 分类排查总流程图

```
发现结构/外观异常
  │
  ├─ 保护壳/防护盖损坏？ ── 运输/装配磕碰
  │     ├─ 已装车 → 客户生产/装配线过程损伤 (#2)
  │     ├─ 未拆包装箱 → 运输/物流损伤
  │     └─ 少量（400件中2件）→ 不支持物流端索赔 (#159)
  │
  ├─ 螺纹孔异常？
  │     ├─ 螺纹孔有残留防锈油 → 偏执性清洗管控，不影响使用 (#118)
  │     ├─ 螺纹孔边缘有漆 → 喷漆工序残留，过丝锥或正常安装 (#214)
  │     ├─ 螺纹孔尺寸厚度差异 → 查图纸范围，符合则放行 (#117)
  │     ├─ 螺纹孔损伤（滑丝）
  │     │     ├─ 通止规检测+硬度测试 → 判断螺栓还是材料问题 (#49)
  │     │     └─ 螺纹错齿 → 客户安装不当 (#129)
  │     └─ 螺纹堵头掉落 → 客户装配时碰掉 (#179)
  │
  ├─ 外观/表面问题？
  │     ├─ 掉漆 → 补漆放行或客户自行喷漆 (#208、#244)
  │     ├─ 喷漆有颗粒/毛刺 → 供应商喷漆前清洁度不足 (#209、#245)
  │     ├─ 生锈
  │     │     ├─ 潮湿/淋雨（限压阀处难喷涂）→ 客户环境导致 (#172)
  │     │     └─ 磕碰划痕后生锈 → 客户工厂责任 (#50)
  │     ├─ 表面油污 → 确认是转向机油还是外部油污 (#226)
  │     └─ 凹坑（壳体砸钢球处）→ 正常制造特征，解释放行 (#225)
  │
  ├─ 缺件/漏装？
  │     ├─ 漏装限位螺栓/螺钉
  │     │     ├─ 检查螺纹端面是否已喷漆、螺纹是否生锈
  │     │     │     └─ 已喷漆+生锈 = 出厂即未装 (#77)
  │     │     ├─ 调取装配线视频记录 → 确认员工是否安装 (#77)
  │     │     └─ 返回修复或客户自行安装 (#77、#97)
  │     ├─ 堵块缺失
  │     │     └─ 检查安装位置有无压痕 → 判断是否漏装 (#38)
  │     └─ 铁丝堵问题
  │           └─ 漏油 → 返回分析（可能为磕碰/来料）(#237)
  │
  ├─ 铭牌/标签问题？
  │     ├─ 铭牌磕碰扫不上码 → 重新生成二维码 (#37)
  │     └─ 不影响转向机功能
  │
  └─ 组件/接插件损坏？
        ├─ 空心螺栓脱落 → 不可修复 (#130)
        ├─ 动力单元X3接头损坏（坠地）→ 不可修复 (#141)
        ├─ X4线束接头损坏（库存）→ 邮寄新件更换 (#152)
        └─ 加长轴输入轴端损坏（转运磕碰）→ 无法现场维修 (#39)
```

## 第二步：排查具体模式的方法

### 保护壳/防护盖损坏排查

**source页实际方法**：
1. 确认损坏程度（#171：损坏一小部分→更换；#2：完全破碎→需更换）
2. 判断损坏时机：未拆包装物流端→已装车客户端
3. 评估影响：长期不更换可能进水/灰尘影响密封性能（#171）

**处理方式**：邮寄保护壳更换即可。转向机本体不受影响。

### 螺纹/螺纹孔问题排查

**source页实际方法**：
1. **螺纹孔有漆/有油**：目视即可确认（#118、#214）。不影响功能，丝锥过一下即可
2. **螺纹尺寸/厚度异常**：查图纸确认公差范围（#117：24-29.5mm以内均为合格）
3. **螺纹滑丝/损坏**：
   - 通止规检测：通规拧上松动、无法卡住螺纹→螺纹已损坏（#49）
   - 硬度测试：确认材料硬度是否一致（#49：未发现材料异常）
   - 判断是否拧错牙：从顶部螺纹判断，拧错牙会有明显起始错位（#49）
4. **螺纹堵头掉落**：查看螺纹处是否已喷漆来判断出厂状态（#179：未喷漆→喷漆时堵头在→客户装配碰掉）

**处理方式**：
- 螺纹孔有漆/油 → 清洁或过丝锥，正常安装
- 客户螺栓错齿/用错螺栓 → 不建议攻丝后使用（#129）
- 堵头掉落/堵块缺失 → 重新提供堵头安装

### 外观/表面问题排查

**source页实际方法**：
1. **掉漆/凹坑**：拍照评估范围和深度（#208、#244、#225）
   - 露铁需补漆防止生锈
   - 凹坑若为壳体制造特征（砸钢球处），解释后即可放行
2. **喷漆颗粒/毛刺**（#209、#245）：
   - 触摸感知是否扎手（#245：不扎手→放行）
   - 与宇通进货检验沟通放行标准
3. **生锈**（#50、#172）：
   - 看位置：侧盖/限压阀处 → 检查对应的装配/存储环境
   - 看方向：正向/难喷涂位置 → 综合判断责任方
   - 检查天气（#172：雨天多，淋雨导致）

**处理方式**：
- 掉漆：补漆放行
- 颗粒/毛刺：提醒供应商注意喷漆前清洁度（#209）
- 生锈：客户工厂责任或淋雨导致，喷漆修复

### 漏装限位件排查

**source页实际方法**：
1. **照片确认**：查看螺纹端面是否已喷漆、螺纹是否生锈（#77：已喷漆→出厂即未装）
2. **视频记录**：调取博世装配线视频记录确认（#77：确认员工未装）
3. **对比确认**：与正常件对比限位件位置和数量

**处理方式**：
- 客户自行安装限位螺栓后即可放行（#77）
- 返回博世修复后使用（#97）
- 需追溯装配线防错系统失效原因

### 组件损坏排查

**source页实际方法**：
1. **RAS空心螺栓脱落**（#130）：查看脱落面状态，判断是否可焊接修复→不可修复
2. **动力单元X3接头损坏**（#141）：视频确认接头+排气孔损坏，为物理坠地导致→不可维修
3. **EHPS X4线束接头损坏**（#152）：展车库存后发现损坏，非使用中损坏→邮寄新件
4. **加长轴输入轴端损坏**（#39）：已装垂臂后转运磕碰→无法现场维修（维修后需上台架检测）

## 第三步：判断责任方

### 博世端责任的情况

- **漏装限位件**（#77产线员工漏装、#97）— 博世装配防错失效
- **喷漆颗粒物**（#209）— 喷漆供方清洁度不足
- **螺纹孔有漆**（#214 8台中4台）— 需PUQ处理

### 客户端责任的情况

- **安装螺栓错齿/用错螺栓**（#49、#129）
- **装配过程磕碰**（#2保护壳破碎、#171防护盖磕碰、#179堵头碰掉）
- **转运/存放磕碰**（#39加长轴损坏、#159磕碰、#172生锈雨天淋水）
- **铭牌磕碰**（#37）— 建议客户装车过程规避

### 双方沟通无争议的情况

- **公差范围内尺寸差异**（#117厚度符合图纸）— 解释后客户接受
- **壳体凹坑（制造特征）**（#225）— 解释后放行
- **掉漆**（#208补漆、#244打磨痕迹）— 解释后同意
- **毛刺不扎手**（#245）— 同意放行
- **生锈（客户环境）**（#50改判为客户责任、#172雨天淋水）

### 需要返回分析的情况

- **铁丝堵漏油**（#237）— 无法手拧动螺栓，需返回分析
- **表面油污**（#226进油口处渗油）— 提交申诉

---

> **一句话总结**：结构装配类问题先看"是否真缺陷"—掉漆补漆、有油有漆清洁后照用、保护壳碎换件、螺纹看通止规。是非对错的关键在证据：拍照留底、查图纸公差、看装配记录。**别一看到壳碎了、有油了、有漆了就直接换总成。**
