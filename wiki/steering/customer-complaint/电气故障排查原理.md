# 电气故障排查：原理一句话说清

## 不工作先看保险 → 最简单最便宜

**原理**：电液转向机（[[entities/EHPS转向机]]）的电动泵需要大电流，[[entities/RAS助力缸]]的动力单元也有供电回路。保险丝是保护电路用的，如果电路短路或过载，保险先断。

**怎么判断**：转向机/后桥完全没反应，仪表盘没故障灯。RAS动力单元接线后烧保险需排查正负极是否接反。

**结论**：查整车保险盒 → 烧了就换 → 省去半天诊断时间。

**典型案例**：
- EHPS保险烧断导致无电助力（#4）
- RAS电源线正负极接反烧保险（#101）

## 亮黄灯先读故障码 → 车自己告诉你怎么了

**原理**：转向机ECU有自诊断功能，检测到异常会记下来并点亮故障灯。读取故障码（DTC）直接告诉你哪里不对，不用猜。

**怎么判断**：仪表盘亮黄色方向盘灯或转向系统灯。

**结论**：用诊断仪读DTC → 按码修 → 不读码就是瞎修。

**注意**：有时仪表亮黄灯但报文中无当前故障码（历史故障），需读取历史故障码后重新上下电复现。另有仪表误报灯的情况（CAN信号源地址未转发），需排查整车CAN信号路由（#173）。

## 电液转向机重新标定 → 恢复出厂角度

**原理**：[[entities/EHPS转向机]]的ECU需要知道方向盘在"正中间"的位置。如果标定丢了（换ECU、断电、拆装后），ECU不知道正中在哪，助力输出就会乱。

**怎么判断**：左右打方向圈数不对称、方向偏、一打就到底。更换转向机或前桥后必须标定。

**结论**：重新标定720° → 问题解决 → 不用换件。

**标定注意**：
- 标定时 **仅上15电，不启动发动机**，否则标定显示"成功"但实际未写入（#25）
- 多台车无法标定时，可尝试多次标定（#112）
- EHPS标定失败也可能是线束供电问题，应先确认X1/X2有电、X3的4号针脚有15电（#184）
- 软限位标定需保证离桥限位1-3mm间隙（#138）

## 无法通讯/亮黄灯 → 先查CAN线和信号

**原理**：RAS产品故障排查中发现，**33%（9/27）的RAS案例是CAN通讯/信号故障**，远高于其他模式。RAS依赖整车CAN信号（车速、转向角、车轮信号等），外部信号中断或错误就会报障。

**怎么判断**：
- RAS不工作 + 亮黄灯，读取故障码如19F0E9（后桥传感器初始化超时）、CAN超时类DTC
- 报文分析发现缺少VDC2、EEC1等必要CAN信号
- 故障码指向外部信号错误而非产品自身

**结论**：
1. 排查整车CAN线及网关信号是否正常
2. 检查CAN信号源是否正确发送（通信矩阵一致性）
3. 排查信号源地址是否已转发到正确网关
4. 如有条件，单连产品模拟CAN输入验证

**典型案例**：
- SAS信号丢帧（#223）、车轮信号错误（#193）、缺少CAN信号（#66）

## 后桥不工作 → 先排查X4线束和X3供电

**原理**：RAS后桥传感器信号通过X4线束传输，X3端口提供工作电源（15电）和CAN通讯。这两个端口是RAS电气故障的高发点。

**怎么判断**：
- 故障码19F0E9（后桥传感器初始化超时）→ **优先排查X4线束**
- 产品无法通讯 → 测量X3端口4号针脚是否有15电

**结论**：
1. 检查X4端口针脚有无变形、退针、虚接、进水（#108、#140）
2. 测量X3端口供电电压（2、3、4号针脚），确认正常24V
3. 排查整车线路是否搭铁（#128）
4. 飞线验证CAN线到转向机端是否正常

**注意**：刷写配置参数用的工具/自连线如果针脚偏大，会撑大X3针脚孔导致接触不良（#228）。

## 智驾退出/外部控制LOCK → 查报文时序

**原理**：[[entities/EHPS转向机]]集成外部控制模式支持智驾系统。如果智驾系统请求时序不当（如上电立即请求外部控制但助力未就绪）、MC信号异常、或软限位未标定，转向机可能进入LOCK或Cancel状态。

**怎么判断**：
- 智驾模式无法进入或行驶中退出
- 报"外部控制LOCK"或"Cancel"

**结论**：
1. 录取完整CAN报文分析请求时序
2. 确认转向机标定已完成（#177）
3. 排查MC信号（数值是否连续滚动）和CRC计算
4. 检查智驾请求外部控制的时机是否满足条件（助力百分比≥20%）（#167）
5. 颠簸路面偶发LOCK需长周期跟踪复现（#178）

## EHPS电机不工作 → 排查供电 + 单连产品验证

**原理**：EHPS电机不工作可能是供电问题、线束问题或电机/泵体内部故障。

**怎么判断**：诊断口无通讯、动力CAN无通讯、转向沉重。

**结论**：
1. 检查OBD口6/14（诊断CAN）和1/9（动力CAN）通讯
2. 单连产品判断是产品端还是整车端问题
3. 逐级排查X1/X2供电、X3针脚电压
4. 确认各接头无虚接退针（#192）
5. 如供电正常但电机仍不工作，考虑更换转向机验证

## RAS液压系统排气 → 正确加注方法

**原理**：RAS液压系统使用单向阀结构，无法用自动加注机加注。系统内有气会导致控制延迟或无法正常工作。

**怎么判断**：更换助力缸/动力单元后，例程控制RAS工作延迟或不工作。

**结论**：摆动轮胎排气，使用手动模式加注（适当提高泵转速）。
