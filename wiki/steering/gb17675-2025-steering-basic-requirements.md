# GB 17675—2025 汽车转向系 基本要求

> Sources: 国家市场监督管理总局/国家标准化管理委员会, 2025-12-02
> Raw: [GB 17675—2025 原文](../../raw/steering/2025-12-02-gb17675-2025.md)

## 概述

GB 17675—2025 是《汽车转向系 基本要求》的最新版本，代替 GB 17675—2021，于 2025-12-02 发布，2026-07-01 实施。适用于 M、N、O 类车辆。本标准是转向系统型式认证和技术要求的基础性法规文件，涵盖转向系统设计、功能安全、试验方法、同一型式判定等方面。

本标准不适用于气压传动转向系统和带自动转向系统的车辆。

## 与 2021 版的主要变化

增加/更改了以下核心内容：
- **新增术语**: 蓄电装置、蓄电性能、蓄电状态、老化效应、能量管理系统、供电装置
- **新增要求**: 能量传输与控制传输共用同一动力源/供电装置时的故障要求（4.3.3.6）
- **新增试验**: 全动力转向系统附加试验方法（5.4）
- **新增条款**: 同一型式判定要求（第6章）
- **新增附录**: 转向电子控制系统功能安全试验报告要求（附录C）、功能安全描述要求（附录D）
- **更严格的功能安全要求**: 全面的文档、验证确认要求（附录B）
- **更改报警信号要求**: 全动力转向车辆报警信号细化

## 核心术语

### 转向系统基本术语

| 术语 | 定义 |
|---|---|
| 转向系统 | 用于改变或保持车辆行驶方向的系统，包括转向操纵装置、转向传输装置、转向车轮和供能装置 |
| 转向操纵装置 | 控制转向工作的操作机构 |
| 转向传输装置 | 在转向操纵装置和转向车轮之间构成功能性连接的零部件组合 |
| 控制传输 | 传送控制信号指令以控制转向装置动作的功能 |
| 能量传输 | 传递能量至转向车轮以控制/调节转向的功能 |
| 供能装置 | 提供能量、控制能量分配及储存能量的装置 |

### 转向系统形式

- **主转向装置**: 主要确定车辆行驶方向的装置，分为人力转向、助力转向、全动力转向三种
- **助力转向**: 转向力来源于驾驶人操纵力和一个或多个供能装置。故障后仍可通过人力完成转向
- **全动力转向**: 转向力完全来源于一个或多个供能装置
- **辅助转向装置(ASE)**: 附加装置，调节转向角以调整转向品质

### 功能安全术语

- **安全概念**: 在系统设计时，为保障系统在故障和非故障条件下仍能安全运行所采取的安全措施
- **电子控制系统**: 通过电子数据处理方式协同实现车辆控制功能的一系列单元的组合
- **安全度量**: 为符合安全目标而给定的具体技术参数的量化值

### 新增电力相关术语

- **蓄电装置**: 储存电能并向全动力转向系统传输装置提供电能的装置
- **能量管理系统**: 监测影响蓄电性能的关键变量（电压、温度、内阻、老化效应等），推断蓄电装置实际能力是否满足要求的系统
- **供电装置**: 为转向系统的蓄电装置提供电能的装置（电池、REESS、发电机、燃料电池等）

## 技术要求核心要点

### 转向系统完好时

- 转向系统应确保轻便性和安全性，在不同车辆类别下操纵力要求不同（M1 类 ≤150N，N3 类 ≤200N 等）
- 转向方向应与行驶方向一致，转向角与操纵装置偏转连续对应
- 转向车轮不应仅是后车轮
- 可与其他系统共用能源，但其他系统故障时转向仍应满足故障时要求
- 转向电子控制系统功能安全要求应满足附录B/C/D

### 挂车要求

- 直线行驶应一致，挂车不应因转向系统影响直线行驶性能
- 规定了 25km/h 和 5km/h 稳态转向的轨迹圆半径差（不大于 0.7m）
- 挂车驶离转向圆外摆值(T)不大于 0.5m

### 失效规定

**一般失效**: 机械部件应为金属或同等特性材质；假设任何时候只发生一个故障

**助力转向系统故障**: 动力源或能量传输故障时，转向角不应突变；超 10km/h 时操纵力应满足要求

**全动力转向系统故障**（核心要求）:

| 故障类型 | 要求 |
|---|---|
| 主转向系统故障(4.3.3.1) | 有限时间内限速至 ≤10km/h |
| 控制传输内部故障(4.3.3.2) | 仍应按完好系统性能转向 |
| 控制传输动力源/供电故障(4.3.3.3) | 10km/h、20m 半径，至少 24 次绕"8"字 |
| 能量传输故障(4.3.3.4) | 至少 25 次绕"8"字；自动减速需故障后 ≥60s |
| 储能水平不足(4.3.3.5) | 发出警告信号 |
| 共用动力源替代方案(4.3.3.6) | 变道要求、自动减速时限、能量管理、报警信号 |

### 报警信号

- **红色报警信号**: 主转向系统故障，可能导致完全丧失转向能力
- **黄色报警信号**: 达不到红色条件的电气自检缺陷
- 能量管理系统相关报警（蓄电性能不足、蓄电状态不满足等分层报警）
- 报警信号符号应符合 ISO 2575:2021 的 J04

## 试验方法

### 汽车试验

| 试验 | 方法 |
|---|---|
| 驶离转向圆异常振动 | M1 50km/h 驶离 R50m 圆周；M2/M3/N 40km/h |
| 回正能力 | 转向角约最大一半，≥10km/h 放开，维持或变大 |
| 完好时操纵力 | 10km/h 进入 R12m 转向圆，≤4s 测量 |
| 故障时操纵力 | 10km/h 进入 R20m 转向圆，≤4~6s 测量 |

### 全动力转向附加试验（新增强调）

- 文档检查（降级策略、电源架构设计）
- 主转向系统故障报警及限速验证（5.4.2）
- 控制传输内部故障的转向性能验证（5.4.3）
- 动力源故障绕"8"字试验（5.4.4）
- 共用动力源替代方案的多项试验（5.4.5）:
  - 制动性能+自动减速
  - 连续变道（M1/N1 18次，其他 9次，90s内）
  - 报警信号检查
  - 能量管理系统评估验证
  - 蓄电性能/状态报警验证
- 能量传输故障绕"8"字试验（5.4.6）

## 同一型式判定

第6章为新增内容，规定了四方面的同一型式判定条件：

1. **转向性能（M/N 类）**: 轮距变化 <5%、轴距相同或减小、转向桥轴荷变化限制、转向系统结构/形式相同等
2. **转向性能（O 类）**: 轮距、轴距、车宽、后悬等参数限制
3. **功能安全**: 转向电子控制系统型号/生产企业/软件版本相同，功能安全描述相同
4. **电磁兼容**: 转向电子控制系统型号/生产企业/软件版本相同

## 附录体系

| 附录 | 内容 | 要点 |
|---|---|---|
| A | 辅助转向装置(ASE)补充规定 | 传动装置要求、故障试验（转向圆+瞬态）、报警信号 |
| B | 功能安全要求（规范性） | 文档要求（提交+备查）、危害分析和风险评估（含全动力转向特殊要求）、安全分析、验证确认、试验方法（含故障注入测试） |
| C | 功能安全试验报告要求 | 文档确认内容记录、验证确认试验记录 |
| D | 功能安全描述要求 | 系统描述、HARA、安全措施等描述内容 |
| E | 液压转向传动挂车特别要求 | 管路性能、供能、保护 |
| F | 汽车列车转向供电特别要求 | 牵引车/挂车要求、故障模拟验证 |

### 附录B 功能安全核心内容（与 EPSCU DTC 关联最紧密）

附录B 危害分析和风险评估总结中规定了以下 3 个安全目标表：

**表B.1 — 转向电子控制系统通用安全要求**（适用于电动助力/电液助力转向）
| 整车危害 | ASIL 等级 | 说明 |
|---|---|---|
| 非预期的侧向运动 | D | 侧向加速度/位移/横摆角速度变化值不超过阈值 |
| 非预期地失去侧向运动控制 | D | 转向操纵力/力矩不超过阈值 |
| 失去助力情况下的转向沉重 | QM 或 A | 转向操纵力/力矩不超过阈值 |

**表B.2 — 全动力转向系统特殊安全要求**
| 整车危害 | ASIL 等级 | 说明 |
|---|---|---|
| 非预期的侧向运动 | D | 增加转向车轮转角变化值安全度量 |
| 非预期地失去侧向运动控制 | D | 增加时间安全度量 |
| 失去转向手感反馈 | B~D | 转向力/力矩不应低于安全阈值 |
| 上下转向执行器不同步 | D | 角度偏差/延迟时间安全阈值 |

**表B.4 — 验证确认测试要求** 规定了针对电动助力转向、辅助转向装置、全动力转向三类系统的故障注入试验，包括供电类、传感器类、电机类、通信接口类故障，在直行、弯道、蛇形工况下的接受准则。

## 与 EPSCU DTC 故障目录的关联

本标准中的「4.3 失效规定」「4.4 报警信号」「附录B 功能安全要求」与 EPSCU 的 DTC 故障码直接相关：

- 全动力转向系统的**故障降级策略**（4.3.3）对应 DTC 中多个故障的消除条件和处理方案
- **报警信号要求**（4.4.2）对应故障灯配置（红色报警/黄色报警）
- 附录B 的**危害分析和风险评估**定义了转向系统的安全目标（ASIL D），是诊断故障设计的顶层依据
- 附录B 表B.4 的**故障注入试验**中供电类、传感器类、电机类、通信接口类故障正好覆盖了 DTC 目录中的主要故障类别
