## 故障诊断分析

### 物理机理分析

**（1）建压即响，不转也响 — 振动源独立于转向动作**

怠速时转向泵转速约 600RPM。叶片泵（8~12 叶片）的输出压力脉动频率约 100Hz，恰好落入转向管柱/方向盘系统的结构模态敏感区（通常 80~150Hz）。

泵的液压脉动 → 高压油管 → 转向阀 → 齿条 → 中间轴 → 转向管柱 → 方向盘。

由于振动源是泵本身的出油脉动，不需要通过转动方向盘施加负载来激发，所以"建压即有"、"不转也响"。

**（2）左右都响 — 排除方向依赖型故障**

转向机内部故障（阀芯阀套不对中、齿条单侧磨损、齿轮齿条间隙不均匀）通常表现为特定方向下症状明显。左右都响说明振动源在转向系统的对称部分——泵。

**（3）转速升高症状减弱 — 共振特征**

```
600RPM  → 异响明显，震感强
  ↑
800RPM  → 异响减弱，变尖锐
  ↑
>1000RPM → 震感微弱
```

泵转速升高后：
- 脉动频率升高至 130~200Hz，偏离转向管柱的共振峰
- 液压管路对高频分量的衰减更大 → 传递到方向盘的振动能量下降

这一"转速-症状负相关"是**共振故障**的典型特征，与机械磨损的"转速越高越响"正好相反。

**（4）方向盘震感 + 转动不顺畅**

压力脉动传递到转向阀，使阀芯与阀套间出现周期性位移波动。驾驶员手力与脉动叠加，手感即为"震撼"和"不顺畅"。

### 排查目标分析

排查应按振动传递路径逐级隔离：

| 目标层级 | 排查对象 | 分析方法 |
|----------|----------|----------|
| 1. 振动源 | 转向泵的液压脉动 | 三条关键现象交叉印证：不转也响 + 转速负相关 + 左右对称 |
| 2. 传递路径 | 高压油管路 | 检查管固定、是否有硬连接点干涉 |
| 3. 承受端 | 转向管柱/方向盘 | 振动加速度测量对比 |
| 4. 下游部件 | 转向机内部 | 博世排查已确认转向机无异常 |

### 相关对手件分析

| 部件 | 相关可能性 | 分析 |
|------|-----------|------|
| 转向机 | **已排除** | 博世现场排查结论，现象特征也不支持转向机内部故障 |
| 转向泵 | **高（根因）** | 换泵后问题解决，所有现象指向泵侧 |
| 高压油管 | 低（可能性低） | 部分案例中管路固定不良会放大脉动，但换泵解决说明泵是本源 |
| 转向管柱 | 低（可能性低） | 管柱是被动传递结构，非振动源 |
| 油壶/回油管 | 低 | 回油侧低压，不会直接传递脉动到手 |
| 发动机悬置 | 低 | 若发动机振动大，换泵不会解决 |

## 诊断结论

- **根因**: 转向泵异常液压脉动，在怠速工况激发转向系统共振
- **判断逻辑**: 不转也响（排除负载依赖型）+ 转速负相关（共振特征）+ 左右对称（排除方向性磨损）→ 三条均指向泵，更换后验证
- **诊断方法**: "踩油门听异响"（转速—症状负相关）是最有力的现场诊断手段
- **措施**: 更换转向泵
- **效果**: 问题解决，车辆放行
