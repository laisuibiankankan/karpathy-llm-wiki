# 亮黄灯不工作案例（序号223）

- **序号**: 223
- **问题类型**: 亮黄灯不工作
- **客户**: Yutong New Energy
- **产品族**: RAS

## 初步处理过程

针对挪威IC15E三桥故障码解析如下：
1.	Steering wheel angle - CAN timeout：无SAS信号或车辆SAS信号出现多周期丢帧；
2.	Steering wheel angle sensor - plausibility check detected errors or not possible：15km/H以下SAS1为主，不校验。15km/H以上SAS1和SAS2转角差值超限值（50°）；
3.	First source of vehicle speed - CAN timeoutt：无EBC2报文或EBC2信号超时；
4.	timeout for vehicle speed message：车速信号超时；
5.	Vehicle speed - plausibility check fail：车辆车速信号出现异常或丢失，检查车辆车速信号；
6.	Software restart without doing shutdown：15和30电上下电时序有问题，导致ECU没有正确关闭；
7.	RASI 1 Message Counter error：RASI MC（计数器）错误；
以上故障均为整车信号问题导致，博世软件同事无法给予软件逻辑的支持，请按照双方技术协议自查，保证整车信号正常发送。
